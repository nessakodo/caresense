# Security Policy

## Supported Versions

| Version | Supported |
| ------- | --------- |
| 0.2.x   | ✅        |
| < 0.2   | ❌        |

## Reporting a Vulnerability

Email `security@caresense.app` with encrypted details (PGP preferred). Please include:

- Steps to reproduce
- Expected vs observed behaviour
- Potential impact assessment

We follow [Coordinated Vulnerability Disclosure](https://www.iso.org/standard/45170.html). You will receive an acknowledgment within 72 hours and regular status updates thereafter.
